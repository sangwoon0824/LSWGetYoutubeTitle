Main 폴더 안에 Main.exe를 실행시키면 됩니다
창에 마우스를 가져다 놓으면 위쪽에 메뉴창이 뜹니다

-실행
  1. 실행 버튼을 누를 시 크롬 창이 뜹니다.
  2. 새로 뜬 크롬창에서 노래를 틀거나 하시면 유튜브 제목을 뽑아와서 출력해줍니다
  3. 처음 실행 시키고 한번 로그인 해서 인증절차 거치야 자동 로그인 기능 사용 할 수 있습니다

-유튜브 로그인
  1. 유튜브 로그인을 눌러 아이디와 비번을 입력하면 다음번 실행시 자동으로 유튜브에 로그인 됩니다
  2. 아이디 비번 틀리면 오류나니까 잘 적어주셈

-권장사항
  1. 프로그램을 종료할때 되도록 크롬창이 아닌 프로그램 자체의 종료버튼을 눌러주세요 알아서 다 꺼집니다
  2. 크롬창 뜰 때마다 소리 최대니까 귀갱 조심 ㅎ
  3. 오류 있거나 불편한점 있으면 말해주세요
